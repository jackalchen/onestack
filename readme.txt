
## 1. deploy OpenStack from scrach.
Only checkout and run it! 
1. Setup a fresh Ubuntu Precise(12.04) OS. 

2. Clone onestack:
svn checkout http://onestack.googlecode.com/svn/trunk/ onestack-read-only

3. run it.
cd onestack && ./oneStack.sh

## 2. delete OpenStack
./delStack.sh

## 3. reset OpenStack
./resetStack.sh clear
./resetStack.sh
